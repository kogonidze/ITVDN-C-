# Текст заданий с сайта ITVDN по четвертому уроку

## Задание №1 
Изменить 12 пример первого урока (работа с документом) и создать общий абстрактный класс для всех частей документа. 

## Задание №2
Создайте класс __AbstractHandler__. 
В теле класса создать методы _void Open()_, _void Create()_, _void Change()_, _void Save()_. 
Создать производные классы __XMLHandler__, __TXTHandler__, __DOCHandler__ от базового класса __AbstractHandler__. 
Написать программу, которая будет выполнять определение документа и для каждого формата должны быть методы открытия, создания, редактирования, сохранения определенного формата документа. 

## Задание №3
Создайте 2 интерфейса __IPlayable__ и __IRecodable__. В каждом из интерфейсов создайте по 3 метода _void Play()_ / _void Pause()_ / _void Stop()_ и _void Record()_ / _void Pause()_ / _void Stop()_ соответственно. 
Создайте производный класс __Player__ от базовых интерфейсов __IPlayable__ и __IRecodable__. 
Написать программу, которая выполняет проигрывание и запись.
